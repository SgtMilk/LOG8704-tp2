# LOG8704-tp2

Pour rouler ce TP2, ouvrir le SampleScene dans Assets/Scenes.

Ce TP a été testé sur Iphone 16 et Iphone 17 pro. Il a été compilé sur MacOS Sequoia et Tahoe.
Certains problèmes de linking on été rencontrés sur Tahoe: il fallait enlever le flag de linking -ld64 sinon l'application ne compile pas. Ce flag est automatiquement généré par Unity, c'est un bug de leur part.

Veuillez utiliser l'image joker.jpeg comme marqueur, imprimé de taille 4x8cm. Il faut simplement pointer la caméra de l'application sur le marqueur dans une pièce éclairée pour voir le tigre.
